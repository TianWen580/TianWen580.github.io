National-level college student innovation and entrepreneurship project

Ecological monitoring project of Chuzhou University’s UAV Aerophotogrammetry Project Technology Center

  • Title: AI-based species identification for biodiversity (as host)
  • Contens:
    • Dataset: Integration of protected and open-source Camera Traps data for biodiversity; in the first phase, each species is selected to participate in live annotation with labeled data; in the second phase, the semi-automatic incremental labeling of the live detector is completed, and the species classification dataset is inferred by the second-phase live detector, finally removing interfering data
    • Two-Stage Recognition: Designing live detection and species classification, where if the confidence level of the classification is below a certain threshold, only the live detection results are provided as feedback
    • Deployment: Completed compression distillation of detection and classification models, and deployed them on Ubuntu environment using Docker
